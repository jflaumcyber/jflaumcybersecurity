Simulated Multi-Vector Attack Network Security 3443

Simulated penetration test against a Costco Wholesale network environment, originating from a compromised end-user desktop in a high-traffic store location. Conducted as part of Network Security 3443 at Oklahoma State University Institute of Technology, November 2023.

Team: Jordan Flaum, Rafey Youniacutt, Mung Mang  ·  Target: Customer/employee data, financial records, and network infrastructure access.

// attack vectors

// attack chain

Kill Chain Walkthrough

  1. Physical drop — Rubber Ducky inserted into an accessible USB port at the member service kiosk, silently launching Evilginx2
  2. Credential capture — When an employee logged in, Evilginx intercepted the session cookie, bypassing PingID 2FA entirely
  3. Initial access — Cookie injected via browser extension; attacker now authenticated as the employee without knowing the password
  4. Lateral movement — Explored shared drives, network shares, and privileged accounts from the compromised session
  5. Privilege escalation — Spear phishing targeted higher-privilege accounts (IT, managers); SQL injection probed public-facing web surfaces
  6. Data exfiltration — Sensitive customer data, financial records, and internal credentials collected
Key finding: PingID 2FA was rendered completely ineffective by the session cookie replay — the attacker never needed to know the employee's password or MFA code.
// recommendations

Remediation Steps

  • Disable USB ports on all public-facing workstations or enforce strict device whitelisting via endpoint policy
  • Hardware MFA (FIDO2/WebAuthn) to resist session cookie hijacking — phishing-resistant by design
  • Advanced email security with DMARC/DKIM/SPF enforcement and sandboxed attachment scanning
  • Parameterised queries throughout all database interactions and regular SQL injection audits
  • Security awareness training with recurring anti-phishing simulations and reporting culture
  • Regular penetration testing and a consistent patch management cadence
