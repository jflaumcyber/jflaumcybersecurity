About the Project Python

A fully playable command-line Blackjack game themed after the Lucky 38 Casino from Fallout: New Vegas. The project demonstrates core programming concepts: conditional logic, loops, functions, state management, and object-oriented design — all wired together into a working game.

The player competes against a dealer, can hit or stand, and wins by reaching 21 or finishing closer than the dealer without busting. Standard casino rules apply throughout.

// features

What It Does

The game handles a full 52-card deck — generating, shuffling, dealing, and scoring. Ace value switching (1 or 11) is implemented to prevent unnecessary busts. The dealer follows standard casino logic: must hit on 16, must stand on 17.

The Ace logic is one of the more interesting challenges: the card dynamically shifts its value from 11 to 1 if the hand would otherwise bust — mirroring how a real casino table operates.
  • Deck generation — full 52-card deck built programmatically with suits and face values
  • Shuffle — randomised using Python's random.shuffle() before each game
  • Ace switching — dynamic 1/11 value based on hand total to prevent busts
  • Dealer AI — follows hard casino rules automatically, no user input needed
  • Win/loss/push detection — all outcomes handled and reported cleanly

Ace-Aware Scoring

Cards are represented as tuples with a face value and suit. Hand totals are calculated with a dedicated scoring function that handles the Ace edge case — counting it as 11 first, then subtracting 10 for each Ace if the total exceeds 21.

# Ace-aware hand scoring def calculate_hand(hand): total = 0 aces = 0 for card in hand: value = card[0] if value == 'A': aces += 1 total += 11 elif value in ['J', 'Q', 'K']: total += 10 else: total += int(value) while total > 21 and aces: total -= 10 aces -= 1 return total

Game Loop

The main loop prompts the player to hit or stand each round, keeps the dealer's hole card hidden until the player stands, then resolves the outcome and reports the result with a clear win/loss/push message.
