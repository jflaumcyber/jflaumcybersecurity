Overview

About the Project

Blackjack Lucky 38 is a fully playable Blackjack game implemented in Python, named after the iconic Lucky 38 Casino from Fallout: New Vegas. The project demonstrates fundamental programming concepts including conditional logic, loops, functions, and object-oriented design.

The game supports a standard Blackjack ruleset: the player competes against a dealer, can hit or stand, and wins by reaching 21 or getting closer than the dealer without busting.

Features

What It Does

The game handles a full deck of cards, shuffling, dealing, and scoring. Special rules like Ace value switching (1 or 11) are implemented to prevent unnecessary busts. The dealer follows standard casino rules — must hit on 16 and stand on 17.

The Ace logic is one of the more interesting challenges: the card dynamically shifts its value from 11 to 1 if the hand would otherwise bust, mirroring how a real casino table operates.

Core Logic

How It Works

Cards are represented as tuples with a face value and suit. A deck is generated programmatically and shuffled using Python's random.shuffle(). Hand totals are calculated with a dedicated scoring function that accounts for Aces.

# Ace-aware hand scoring def calculate_hand(hand): total = 0 aces = 0 for card in hand: value = card[0] if value == 'A': aces += 1 total += 11 elif value in ['J', 'Q', 'K']: total += 10 else: total += int(value) while total > 21 and aces: total -= 10 aces -= 1 return total

The game loop prompts the player to hit or stand each round, displays the dealer's hidden card only after the player stands, then resolves the outcome and reports the result.
