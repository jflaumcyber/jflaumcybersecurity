Single Node Installation ongoing

Wazuh is an open-source SIEM platform used for threat detection, integrity monitoring, and incident response. This setup documents deploying a single-node instance from scratch.

Step 1 — Run the install script:

# Download and execute the Wazuh installer curl -sO https://packages.wazuh.com/4.12/wazuh-install.sh && \ sudo bash ./wazuh-install.sh -a

Step 2 — When the script finishes, note down the admin password shown in the terminal output.

Step 3 — Open a browser and navigate to the IP address or DNS name of your Wazuh VM. The dashboard should load and give you access to your SIEM environment.

Step 4 — Log in with username admin and the password from the terminal output.

Syslog Decoders & Rules

Decoders and rules for syslog integration are available at github.com/object1st/wazuh-examples. Instead of modifying the defaults, create separate files for cleaner organisation.

Configuring Syslog Reception

Add the following block to /var/ossec/etc/ossec.conf under the closing </ossec_config> tag:

<!-- ossec.conf syslog receiver block --> <remote> <connection>syslog</connection> <port>514</port> <protocol>udp</protocol> <allowed-ips>192.168.0.0/24</allowed-ips> <allowed-ips>127.0.0.1</allowed-ips> </remote>
Note: Update allowed-ips to match your actual subnet before restarting.
# Restart the manager and verify the port systemctl restart wazuh-manager ss -tunlp

VirtualBox Installation

Kali Linux is a Debian-based distribution purpose-built for penetration testing and security research. You may need to enable virtualisation in BIOS/UEFI (Intel VT-x / AMD-V) before proceeding.

  • Open VirtualBox → Machine → New
  • Name & OS: Set Type to Linux, Version to Debian (64-bit)
  • Memory: Minimum 2048 MB — more RAM means better tool performance
  • Hard Disk: VDI format → Dynamically allocated → 80 GB
  • Click Create to finish the wizard

VM Settings

SectionSetting
General → AdvancedShared Clipboard: Bidirectional · Drag'n'Drop: Bidirectional
System → MotherboardBoot Order: Hard Disk (1st), Optical (2nd), rest disabled
System → ProcessorProcessors: 2 · Enable PAE/NX: ✅
Display → ScreenVideo Memory: 128 MB · Accelerated 3D: ❌ Disabled
Warning: Accelerated 3D graphics causes known display issues — keep it disabled.

First Boot & Install

  • Press Start → click the folder icon to mount your Kali ISO
  • In Optical Disk Selector: Add → select your ISO → Choose → Start
  • Proceed through the Kali installation wizard as a normal bare-metal install
  • The wizard auto-detects the VM and installs VirtualBox Guest Additions

To expand storage later: power off → File → Tools → Virtual Media Manager → select the VDI → adjust size → Apply → use gparted inside Kali to extend the partition.

// ubuntu vm

VirtualBox Installation

Ubuntu is a beginner-friendly Linux distribution ideal for general-purpose use, development, and learning. Download the ISO from ubuntu.com/download/desktop and VirtualBox from virtualbox.org/wiki/Downloads.

  • Click New — include "Ubuntu" in the name to auto-fill Type and Version
  • Point ISO Image to your downloaded file — leave unattended install enabled
  • Set credentials: default is vboxuser / changeme — change these or you'll get a user without sudo access
  • Enable Guest Additions for dynamic screen resizing and clipboard sharing

Post-Install

The unattended installer runs automatically on first boot — don't interact with the initial prompt. The machine reboots when done. Log in then run:

sudo apt update && sudo apt upgrade -y sudo snap refresh
